Configuration secrète

Le mot de passe, les rotors et le plugboard doivent être identiques pour déchiffrer.

Important : cette démonstration chiffre avec l'API Web Crypto. Aucun message ni mot de passe n'est envoyé à un serveur.

Message

Écris un texte clair, puis chiffre-le. Tu peux aussi coller un paquet chiffré pour le déchiffrer.

Sortie

Le résultat contient la version, le sel, le vecteur d'initialisation et le texte chiffré.
